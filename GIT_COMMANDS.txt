# Git Commands to Commit Your Project

# Step 1: Initialize Git repository (if not already initialized)
git init

# Step 2: Add remote repository
git remote add origin https://github.com/2022cs668-Waleed/Mess-management-system.git

# Step 3: Add all files to staging
git add .

# Step 4: Commit with message
git commit -m "Initial commit: Complete Mess Management System with Render deployment configuration"

# Step 5: Set main branch
git branch -M main

# Step 6: Push to GitHub
git push -u origin main

# If you get authentication error, use:
# git push -u origin main --force (only if repository is empty)
